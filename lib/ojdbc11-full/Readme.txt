ojdbc11-full.tar.gz - JDBC Thin Driver and Companion JARS
========================================================
This TAR archive (ojdbc11-full.tar.gz) contains the 23.5.0.24.07 release of the Oracle JDBC Thin driver(ojdbc11.jar), the Universal Connection Pool (ucp11.jar) and other companion JARs grouped by category. 

(1) ojdbc11.jar (7199197 bytes) - 
(SHA1 Checksum: 6e07003d57be5e280ef0492209e20fa27edec5b6)
Oracle JDBC Driver compatible with JDK8, JDK11, JDK12, JDK13, JDK14, and JDK15.
 
(2) ucp11.jar (1539742 bytes) - (SHA1 Checksum: 5ea8cd08e0a7f43255497a6ad332a20f463aa3b8)
Universal Connection Pool classes to be used with ojdbc11.jar -- for performance, scalability, high availability, sharded and multitenant databases.

(3) rsi.jar (260432 bytes) - (SHA1 Checksum: b3fdb0b2fcb6555fd3eedb259c1a6bd9ec079fb9)
Reactive Streams Ingestion (RSI) 

======================
Security Related JARs
======================
Java applications require some additional jars to use Oracle Wallets. 
You need to use all the three jars while using Oracle Wallets. 

(4) oraclepki.jar (501935 bytes) - (SHA1 Checksum: a99959feb6995c8e079cd6e658ac6fb040e3fbdd)
Additional jar required to access Oracle Wallets from Java
